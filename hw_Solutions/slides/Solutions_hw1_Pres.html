Render the document

Via R Studio

Via code

If you don't have the rmarkdown library installed, first run

install.packages("rmarkdown")

Then, to render to document

library(rmarkdown)
render("yourFile.Rmd", "html_document")

Note that your working directory must be set to the same location as the .Rmd file, or you can provide a path to the file in the render function.
